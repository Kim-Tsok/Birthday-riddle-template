# Riddle Birthday Website (Template)

A customizable riddle-based birthday website built with React and Tailwind CSS.

This repository is intended to be **forked** and personalized. It is not a hosted service. You copy it, change the content, deploy it, and surprise someone

---

## What is this?

This is a template for creating a fun birthday surprise where the recipient solves riddles to unlock the final message.

It was originally made as a personal gift, but has been refactored to be reusable for **anyone**.

---

## How to Use (Important)

**This project is meant to be forked.**

### Step 1: Fork this repository

Click the **Fork** button at the top right of this page.

This creates your own copy of the project.

---

### Step 2: Customize the content

You can change:

- Birthday messages
- Riddles and answers
- Any text shown on the website

No need to rewrite components unless you want to

---

### Step 3: Run locally

```bash
npm install
npm run dev
```

**OR**

Deploy it on vercel, netlify, github pages or any means you see fit
